# 配管切り寸法計算アプリ

JIS B 2301 準拠、ねじ込み式管継手のパイプ切り寸法を計算するWebアプリ（iPhone対応）。

## 機能

- **直線配管（90°系）** モード: `切り寸法 = 芯々 − G − G`
- **45°オフセット** モード: `切り寸法 = 芯々 × √2 − G − G`
- **捻り（前後のズレ）** 対応: 三平方の定理で合成 `√(芯々² + 捻り²)`
- 対応口径: 1/2〜4インチ（JIS B 2301）
- 異径接続対応

## 使い方

ブラウザで `index.html` を開くだけ。iPhoneのSafariでも動作します。
